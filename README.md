🧬 DNA Analyzer Pro

A powerful bioinformatics web application built using Streamlit that allows users to analyze DNA sequences, detect ORFs, perform BLAST searches, and explore gene data from NCBI.

---

🚀 Live Demo

👉 https://dna-analyzer-smitangshu.streamlit.app

---

🔬 Features

- 🧬 DNA Sequence Analysis
  
  - Length calculation
  - GC Content
  - Nucleotide distribution graph

- 🔁 Transcription & Translation
  
  - DNA → RNA
  - RNA → Protein

- 🧠 ORF Detection
  
  - Finds Open Reading Frames
  - Highlights coding regions

- 🧪 BLAST Search
  
  - Real NCBI BLAST integration
  - Fake BLAST fallback system

- 🔍 NCBI Gene Search
  
  - Search genes (e.g., BRCA1)
  - Direct links to NCBI database

- 📄 PDF Report Generation
  
  - Includes full analysis
  - Watermark + border
  - BLAST + NCBI results

---

🛠️ Tech Stack

- Python 🐍
- Streamlit

- ReportLab

💡 How to Use

1. Enter a DNA sequence
2. Click Analyze Sequence
3. Explore:
   - DNA Analysis
   - BLAST Results
   - NCBI Gene Search
4. Download full report as PDF
